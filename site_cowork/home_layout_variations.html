홈 화면 구성안 4종

같은 콘텐츠를 ‘무엇을 가장 먼저 보여줄까’ 관점에서 4가지 흐름으로 재배치한 비교 시안입니다.

A안

정통 베이커리

~ 안정적인 첫인상 ~

큰 히어로 → 매장 소개 → 카테고리 → 인기 슬라이더. 기존 PARKHAD와 가장 유사한 안정적 구성.

✓ 신뢰감 ✓ 정보량 ✓ 보편적
B안

갤러리 중심

~ 인스타 감성 ~

작은 슬로건 후 곧장 큰 케이크 그리드. 비주얼이 가장 먼저 ‘맛있어 보이는’ 인상을 전달.

✓ 시각 임팩트 ✓ MZ 타깃 ✓ 빠른 둘러보기
C안

스토리텔링

~ 마음을 건드려요 ~

"오늘은 누구를 위한 케이크?" 시나리오 카드와 손글씨 후기로 ‘선물’의 감성을 강조.

✓ 정서적 공감 ✓ 선물 시장 ✓ 차별화
D안

예약 우선

~ 바로 예약해요 ~

상단부터 큰 예약 CTA와 4-Step 미리보기. 광고로 들어온 사용자를 빠르게 전환.

✓ 전환율 ✓ 광고 랜딩 ✓ 명확한 행동
