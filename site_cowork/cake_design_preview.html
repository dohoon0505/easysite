1. 무드 변화

Before → After

BEFORE · PARKHAD

남성 헤어샵
  • 다크 모드 기본 + 모노 그레이 팔레트
  • 골드(#C9A961) 단일 액센트, 절제된 사용
  • Pretendard 900 / 헤비 트래킹
  • 각진 카드, 시크하고 묵직한 톤
  • "간지나고 단단한" 남성적 인상

AFTER · 레터링케이크샵

달콤하고 사랑스러운 케이크샵
  • 라이트 모드 기본 + 주황 + 크림/아이보리/코랄
  • 주황 #F58634을 메인으로 풍성하게 활용
  • Pretendard + Jua + Gaegu(손글씨) 믹스
  • 둥근 모서리(16-24px), 부드러운 그림자
  • "귀엽고 러블리한" 베이커리 무드

5. 컴포넌트 키트

버튼/태그/입력/카드

버튼 (Buttons)

모서리는 모두 pill(9999px). 메인은 orange + soft shadow로 떠 있는 느낌.

태그 (Tags)

BEST 💖 NEW 시즌한정 SALE
BEST는 코랄, NEW는 메인 주황. 시즌한정은 아이보리에 잉크 컬러.

입력 필드 (Input)

포커스 시 주황 외곽선 + soft glow. 둥근 모서리(16px).

케이크 카드 (Catalog)

컬러풀 생일 케이크
35,000원~
썸네일 위 BEST/NEW 배지. 카드 hover 시 살짝 떠오름(translateY -2px).

예약 진행 표시 (Stepper)

1. 디자인 2. 날짜 3. 문구 4. 확인
현재 단계는 풀컬러, 완료 단계는 soft, 미진입은 dashed.

안내 배너 (Info)

🧁최소 3일 전 예약을 권해드려요
파스텔 주황 배경 + 점선 테두리. 이모지/아이콘 1개 동반.

  • 카테고리 탭 (생일/기념일/웨딩/미니)

  • BEST · NEW · 시즌한정 배지
