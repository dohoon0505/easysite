04. 사용 비율 · 60-30-10 원칙

컬러는 ‘양’이 ‘인상’을 결정합니다. 주황을 적게 써야 더 강하게 보입니다.

60-30-10
~ 비율의 약속 ~
Cream / Ivory62%
베이스. 모든 배경과 카드 표면. 가장 넓은 면적.
Ink / Text22%
텍스트와 라인. 본문 가독성을 책임지는 두 번째 면적.
Orange (Primary)12%
CTA · 로고 · 강조. ‘시선이 머무는 곳’에만 사용.
Coral (Accent)4%
BEST · NEW · 진짜 강조 포인트만. 4% 초과 금지.
💡 주황을 50% 이상 칠하지 않습니다. 면적이 커지면 ‘오렌지 가게’가 아니라 ‘오렌지 광고’처럼 보입니다. 둥근 모서리 + 부드러운 그림자로 ‘떠 있는 버튼/카드’에 집중하세요.

06. 접근성 · 대비 검증

WCAG 2.1 AA (4.5:1 일반 텍스트 / 3:1 큰 텍스트) 기준으로 검증한 컬러 조합.

샘플전경 / 배경대비비일반 텍스트큰 텍스트 (18pt+)UI 컴포넌트
예약하기 #FFFFFF / #F58634 4.06:1 ⚠ 작은 본문 X ✓ Pass ✓ Pass
예약하기 #FFFFFF / #E26A18 4.93:1 ✓ Pass ✓ Pass ✓ Pass
본문 텍스트 #3B2A21 / #FFF9F2 12.8:1 ✓ AAA ✓ AAA ✓ Pass
부 텍스트 #5C4538 / #FFF9F2 8.4:1 ✓ AAA ✓ AAA ✓ Pass
강조 텍스트 #E26A18 / #FFF9F2 4.5:1 ✓ Pass ✓ Pass ✓ Pass
BEST #FFFFFF / #FF8B7A 2.7:1 ✗ Fail ⚠ 14pt+ 굵게만 ✓ Badge OK
Secondary #B8520E / #FFE9D6 5.8:1 ✓ Pass ✓ Pass ✓ Pass
⚠️ 주황(#F58634) 위 흰 텍스트는 14px 이하 본문 금지. 16px 굵기 700 이상에서만 사용. 일반 본문 강조가 필요하면 --brand-hover (#E26A18)를 텍스트 컬러로 쓰세요.
⚠️ 코랄(#FF8B7A)은 배지 전용. 본문 텍스트 컬러로 사용 금지. 위 흰 글자 ‘BEST’ 같은 굵은 짧은 라벨에서만 OK.

07. DO / DON'T

자주 발생하는 실수와 권장 패턴.

✓ DO · 권장
예약하기
큰 주황 버튼 + 흰 글자 (메인 CTA)
상세보기
연한 주황 배경 + 진한 주황 글자 (보조 액션)
카드 콘텐츠
크림 배경 + 잉크 텍스트 (가독성 12:1)
BEST
코랄은 작고 굵은 배지로만 사용
✗ DON'T · 금지
잉크 텍스트
주황 위 잉크 텍스트 (대비 2.1:1, Fail)
전체 배경을 주황으로
주황을 넓은 면적에 칠하지 않음 (12% 룰)
검정 배경
다크 배경 사용 금지 (음식 사진 채도 손실)
긴 본문 텍스트를 코랄로
코랄을 본문/제목 컬러로 쓰지 않음

color_tokens.css
/* CSS Custom Properties */ :root { /* Primitive · Orange */ --orange-500: #F58634; --orange-600: #E26A18; /* Semantic */ --brand-default: var(--orange-500); --bg-default: #FFF9F2; --text-primary: #3B2A21; ... }
color_tokens.json
{ "brand": { "primary": "#F58634", "hover": "#E26A18", "subtle": "#FFE9D6" }, "background": { "default": "#FFF9F2", "surface": "#FFFFFF" }, ... }
